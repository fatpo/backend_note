cat 作用：
```
显示某个文件的内容到终端。
```

举个例子，我们把"hello world"写入 "test.txt"：
```
~ > echo "hello world" > test.txt           
```

然后用`cat`命令看看文件`test.txt`内容：
```                                                                                        
~ > cat test.txt                                                                                                                   
hello world
```
