## less
类似 `vi`和 `cat`，也是查看文件内容的。

不过它支持一些互动操作，可以翻页什么的，比较方便。

## 演示

查看 less 的用法：
```
man less
```

上面用法很多，不过我们经验来看，用的最高频就`翻页`和`上下行`两个操作。

翻页操作：
```
ctrl + F - 向前移动一屏
ctrl + B - 向后移动一屏
ctrl + D - 向前移动半屏
ctrl + U - 向后移动半屏
```
话说 f 是 front，b 是 back，这个 d 和 u 是什么缩写呢。

上下行操作：
```
j - 下一行
k - 上一行
```

和 `vi` 一样，按 q 退出。
